# Workspace: Minor Devs Studios Website

This workspace contains the personal portfolio/studio website for **Minor Devs Studios** — a solo developer studio brand.

## Project Location

The active project lives in `cards-web/`. That's where all source code, components, and configuration reside.

```
CW/
└── cards-web/      ← main Next.js project (work here)
```

See `cards-web/CLAUDE.md` for the full project context, completion status, goals, and progress tracking.

## Quick Dev Commands (run from `cards-web/`)

```bash
npm run dev       # start local dev server at localhost:3000
npm run build     # production build
npm run lint      # run ESLint
npm run typecheck # TypeScript check (tsc --noEmit)
```

## Deployment

Hosted on **Vercel**, auto-deploys on push. Domain: **minordevstudios.com**
