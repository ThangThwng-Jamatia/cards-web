## Customizing Your Domain

After deploying to Vercel, update the site URL:

1. **Option 1: Environment Variable** (Recommended)
   - Go to your Vercel project settings
   - Add environment variable: `NEXT_PUBLIC_SITE_URL=https://your-domain.com`

2. **Option 2: Direct Update**
   - Edit `app/layout.tsx`
   - Change line 9: `const siteUrl = 'https://your-custom-domain.com'`

## Troubleshooting

### Favicon not updating?
1. Hard refresh: `Ctrl+Shift+R` (Windows) or `Cmd+Shift+R` (Mac)
2. Clear browser cache
3. Check browser DevTools → Network tab for favicon requests

### Wrong domain in previews?
- Update `NEXT_PUBLIC_SITE_URL` in Vercel environment variables
- Redeploy the site
